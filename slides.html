💪 Fitness Tracker CLI Project
By Samuel Karobia 👨🏽‍💻
📌 About This Project
The Fitness Tracker CLI is a command-line application developed in Python that allows users to register, manage their profiles, and track their workouts and exercises. It supports adding, updating, and deleting entries with a clean and interactive CLI interface. Built with SQLAlchemy and SQLite, it demonstrates strong understanding of object relationships and database integration.
📋 Project Overview
A Command-Line Interface (CLI) app to track users, workouts, and exercises using Python and SQLAlchemy ORM.
✨ Key Features
  • 🧍 User registration and management
  • 🕒 Workout logging with time and date
  • 🏋️ Exercise tracking (reps, sets, type)
  • 🗃️ Data persistence with SQLite
  • 🛠️ Full CRUD operations from CLI
🛠️ Technologies Used
  • 🐍 Python 3
  • 📦 SQLAlchemy ORM
  • 💾 SQLite Database
  • 🧪 Pipenv for dependency management
📚 Challenges & Learnings
  • 🔗 Designing SQL relationships (One-to-Many, Many-to-One)
  • 🖋️ Improving user experience through formatted CLI outputs
  • 🔍 Debugging and testing functions end-to-end
🏁 Conclusion
A practical and scalable fitness tracker solution built entirely with Python, showcasing object-oriented programming and database integration.
